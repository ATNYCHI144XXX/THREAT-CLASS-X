## Research Protocol

### New Anomaly Documentation
When adding new locations, include:
1. Precise GPS coordinates (decimal degrees)
2. Date of observation
3. Screenshot/image evidence
4. Description of unusual features
5. Proposed explanation/hypothesis
6. Verification status (confirmed/unverified)
7. Magnetic field data if available
8. Access restrictions if any

### Classification System
- **Type A**: Verified anomalies with unknown origin
- **Type B**: Likely natural formations with unusual properties
- **Type C**: Confirmed artificial structures with unclear purpose
- **Type D**: Unverified claims requiring investigation
- **Type X**: Explained phenomena (archive for reference)
